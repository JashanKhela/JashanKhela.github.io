Portfolio

SpotSharer:   SpotSharer is an Airbnb like web application made for users to be able to rent out their parking spots in major cities in BC. You can signup or login using your Facebook or Google account. As a user, you are allowed to either rent a parking spot or host your own spot that you are not using. From there you can view the parking spots that you have rented, with the ability to add and remove vehicles to the spot. You can add details such as plate number and make to ensure the safety of your vehicle. As a host of a spot, you can send direct messages to the users renting out your spot to provide them with any further details as well as images of the parking spot. The back-end is powered by NodeJS while the front-end was built using ReactJS and Bootstrap. Check out the project on my GitHub.

Mind Of Mine:  A personal project that is very important to me. The Mind of Mine Foundation is a non-profit organization based around spreading awareness around mental health. As one of the founders, I hold this organization very close to my heart. I wanted to make sure everyone knew about it and had a place to go to see upcoming events as well as the money raised. I took the task of creating the website for Mind of Mine from scratch using ReactJs and GatsbyJS. Some of the features include making donations, tracking donations, and integrating our Instagram feed. Check out Mind of Mine!.

Smart Cat:  A smart auto categorizing to-do list. How does it work you ask? Simple input any simple task such as "Time magazine" and our list will automatically add " Read Time Magazine " to your list of things to do. You can also change the task if it ever gets guessed incorrectly as well as delete tasks that have been completed or that you want to be removed. SmartCat was built using NodeJS and Express for the back-end. For database migrations and querying data, we used Knex and Postgresql. The front-end was built using Bootstrap, XML and SASS. Check out the project on my GitHub.

Chatty App  ChattyApp is a web-chat application built using ReactJS and Websockets. The main purpose of this app is to provide users with a real-time chat application to talk to fellow users in the chat room. Some of the features include connecting to a chat room, viewing the number of users online and viewing posts by other users. Check out Chatty App here.

Tweeter :  Tweeter is a simple, single-page Twitter clone. Tweeter is here to help you share your experiences with others and help get you connected to others like you. With Tweeter you can compose a new tweet and view other tweets. The front-end was built using HTML and CSS while the back-end was built using Express and NodeJS. We used MongoDB to store the tweets. Check out Tweeter here.
